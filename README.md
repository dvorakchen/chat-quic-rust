# chat-quic-rust

a simple chat with QUIC in rust, just for practice, not completed yet

only have the echo ability

# Usage

you need two terminal, one of terminals as server

```sh
# server
$ cargo r -- -c <your certificate> -k <private key> -l 127.0.0.1:4433
```

```sh
# client
$ cargo r
$ <your email>
```
